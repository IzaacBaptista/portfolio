Conhecimentos

HTML

É uma linguagem de marcação de hipertexto. O HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web..

CSS

É uma linguagem de folhas de estilos usada para definir o aspecto geral e a formatação (aparência) de um documento que está escrito em uma linguagem de marcação.

UI/UX

O UI/UX design preocupa-se com os fluxos e a estrutura da informação apresentada. Responsável por criar protótipos e definir objetivos e passos sobre aquela que será a melhor experiência do usuário.

Javascript

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.

Boostrap

Bootstrap é um framework CSS para ser utilizado no front-end de aplicações web. Ele utiliza JavaScript e CSS para estilizar as páginas e adicionar funcionalidades ao site.

Git e GitHub

GitHub é um serviço baseado em nuvem que hospeda um sistema de controle de versão chamado Git
